# Виртуальный фон для веб-камеры

Этот проект основан на Vite + Vue 3 и включает предпросмотр виртуального фона с сегментацией изображения.

## Запуск проекта

```bash
npm install
npm run dev
```

После запуска откройте SPA (по умолчанию http://localhost:5173) и перейдите к предпросмотру виртуального фона.

## Как это работает

SPA обрабатывает видеопоток в `<canvas>` при помощи MediaPipe и сегментации фона.
